# EtyaMod 故事背景设定

> 本文档整理自 idea.txt(两年前与 AI 的对话记录),已解决其中的命名冲突和设定矛盾,作为后续开发的世界观基准。
> 模组暂定名:《余烬之梦(Ashes of a Dream)》

---

## 一、世界观总览

这不是《泰拉瑞亚》的续写,而是对"已经结束的神话"的解构。

- 玩家扮演的**不是英雄**,而是一个晚来的普通人,在没有克苏鲁、没有月球领主的平静世界里,寻找"意义"与"遗产"。
- 核心主题:**遗忘、记忆、重构**;"英雄遗产 vs 新时代挣扎者"。
- 最终要面对的,是那个已经超越常人的"传说本身"——手持天顶剑的前代主角。

## 二、世界的两层真相

### 表层(玩家开局所见)

- 月球领主被击败多年,神明、怪物、灾厄都已沉寂。
- 人类重建家园,腐化、猩红等危险区域被"净化"成资源区。
- 前代主角成为传说,被后人称为**"终焉者"(The Endbringer)**,被描绘成英勇、坚定、神明在人间的代理。

### 深层(剧情逐步揭示的真相)

- 终焉者击败月球领主、终结克苏鲁信仰的实体化支配,但代价惨重:
  - **城镇 NPC 其实是"旧日意识形态的具象"**——寄居于人类意识中的伪装。克苏鲁崩解后,他们失去依托,理智、形体、存在开始裂解、异变。
  - 终焉者陷入"终焉之后的清算":曾经亲密的盟友,一个个变成他不得不亲手斩杀的怪物。
- 他并非堕落,也非愤怒,而是**"全然冷静地毁灭"**——"我曾亲手建起这座城……如今也只有我能将它埋葬。"
- 最终他用尽最后之力,**将天顶剑封印进净化核心,把整片大陆封印成一个稳定的"和平幻象"**。后世所居的世界,其实是假的。
- 历史由幸存者书写,但没有人幸存下来说出真相——于是传说成了神话,真相成了禁忌。他真正的名字被遗忘,只留下"终焉者"这个称号。

### 幻象的运作与崩坏(Boss 框架的世界观基础)

- 幻象世界用"那段神话的记忆"编织,因此**神话必然重演**——这是世界的规则,不是某人的阴谋;向导只是规则的执行端。
- 幻象开局是"完美的和平":**星星不坠落、血月不升起**(玩家梦中低语的由来)。魔力水晶靠陨落之星合成,所以这个时代**魔法失传**;旧时代的魂灵无法被唤起,所以**召唤失传**。
- 玩家是幻象里的异物:体内碎片觉醒度越高,周围幻象越薄,旧日噩梦就在薄处"重演"。**肉山前的原版 Boss 都是幻象回放的"回响"**——这解释了它们可反复召唤重打(记忆可以回放,真实的怪物不能复活两次),也解释了它们掉落记忆碎片(它们本身就是记忆)。
- **肉墙是封印的守门者**,是幻象表层与深层的边界。击败肉山 = 杀死"第一句谎言"的本体,幻象表层大规模撕裂——**困难模式的神圣/腐化扩散,蔓延的不是邪恶,是真实**(幻象崩坏的斑块),爆发的幻象力量随之入侵现实。
- **肉山后的 Boss 是真实存在**:机械三王是终焉者战争兵器的记忆借入侵之力实体化,万物价主、血医魔母等旧 NPC 异变体也在此时重拾力量、盘踞现实。
- **月球领主是封印核心里最沉重的那段记忆**——终焉者最后一战的记忆质量太大,幻象撕裂后它是最后一个、也是唯一能完全实体化的回响。玩家击败月总 = 亲身重演终焉者的最后一战,而且这次是真的;打完这一战,玩家才第一次"亲眼看见"真相。

## 三、核心角色

### 1. Etya(艾缇娅)——唱歌的引导者

| 项目 | 设定 |
|------|------|
| 身份 | 来自旧时代的"幸存之音"/"残响"——终焉者封印世界时亲手留下的**记录者**,他给她的指令只有一条:记录。她的幸存、记忆残缺、唱歌忘词皆源于此(记录载体随幻象一同磨损,是幻象衰亡的征兆) |
| 与终焉者的关系 | 曾是他的旅伴与战斗支援者(可暧昧处理为恋人)。大战后英雄隐去,她选择留在人间,等待"下一个可能改变一切的人" |
| 外观 | 白发、披灰绿披巾、双眼如雾,背着一把类似竖琴的弦乐器 |
| 初登场 | 荒废旧镇边缘,坐在废弃井旁的石椅上,轻唱无词之歌 |
| 语气 | 柔和、诗意,像只活在"梦与残响"中的人;记忆残缺,唱到一半会忘词 |
| 功能 | 剧情引导、出售旧时代遗物、记忆碎片兑换、开启梦境副本 |

**她引导玩家的动机(三层递进,也是她逐步向玩家坦白的三个阶段):**

1. 她是"时代的遗音",使命是把遗失的记忆传递给值得继承的人。她的歌词其实是旧 NPC 的遗言、Boss 的哀鸣、终焉者的独白碎片。
2. 她能"听见玩家体内的回响"——玩家出生时沾染了旧时代的某个碎片(残破武器/旧剑鞘/古书页,具体待定),这是她起初愿意引导的原因,也是玩家力量成长的叙事来源。
3. 她知道玩家终将遇到那位"沉眠的战神",但不阻止;她只希望悲剧不要重演,试图让玩家用别的方式成长。若玩家走向暴力路线,她会悲伤地停止唱歌(Bad Ending 分支前兆)。

**深层动机(有私心的守望者,记忆之井或终章前才揭开):**

- 终焉者给她的指令只是"记录",但她**自己决定越出指令**:虚假和平对终焉者不是安息,而是永无止境的独自守望——他被自己造的封印囚禁在世界边缘。要让他解脱,只有培养出一个足以站到他面前的人;而培养的唯一方式,就是让神话重演一遍。
- 因此她知道向导会去动树妖之血、知道幻象终将撕裂,却**选择沉默**。树妖之血事件不是她的本意(那是"剧本"/向导所为),但她的不作为是共谋。她的温柔是真的,她的算计也是真的。
- 她对暴力路线玩家的悲伤由此有了具体来源:她怕自己亲手造出第二个终焉者,怕剧本的结尾还是老样子。
- 肉山之后,树妖会发现 Etya 当时知情,两人有一场**对质**(中期揭示 Etya 深层动机的演出节点)。

**结局相关**:她最终献出自己,化为一段旋律,唤醒沉眠中的传说英雄(引出最终战)。这不是突然的牺牲,而是**赎罪**——灾难是她放进来的,代价她来付。

### 2. 终焉者(The Endbringer)——最终 Boss

| 项目 | 设定 |
|------|------|
| 身份 | 前代玩家主角,已"超越人类",宛如神明 |
| 外观 | 身披披风、银发,站立不动时仿佛雕像,背负天顶剑 |
| 性格 | 冷漠、近乎机械;只在最终一战时展现内心火焰 |
| 现状 | 孤身守在世界边缘(净化核心/"记忆的尽头"),守望自己造出的虚假和平 |
| 战败后 | 化为灰烬,只留下一句:"不要成为我。" |

**战前台词:**
- "你终于来了。你不该来的。"
- "我是遗产,不是敌人。但你要的,是一个可以超越的神。"
- "那么,就让我,见证你的火焰。"

### 3. 异变旧 NPC(剧情 Boss)

旧 NPC 失去克苏鲁依托后的异变形态。**注意:三者的存在形式不同**(详见第五节"Boss 框架"):

- **向导 → 虚言者·卡诺斯(幻象,梦境副本 Boss,肉山前)**:旧神残影的一部分,"剧本的执行者"——旧主的剧本要求神话重演,他负责推动(树妖之血事件即他所为)。他与肉墙同源,是墙留在人间的"发言人";最后会说:"你以为是我指引了你,其实……我只是重复旧主的剧本。"打败他获得**"虚言者的傀儡"**,扔入熔岩召唤肉山(原版"杀死向导召唤肉山"的剧情化)。
- **商人 → 贪欲之灵"万物价主"(真实存在,肉山后)**:嘴裂至耳根,身体中藏满无尽物品,满嘴"交易、等价、剥夺"。幻象撕裂后借入侵现实的幻象力量重新实体化,盘踞于现实世界的巢穴。
- **护士 → 血医魔母(真实存在,肉山后)**:执着于"完美体液构造"的肉体魔女,不断把伤口掰开、愈合、再掰开。同为实体化后重拾力量的真实存在。
- (其余 NPC 异变形态可按需追加)

### 4. 树妖——半清醒的封条

| 项目 | 设定 |
|------|------|
| 身份 | 旧日意识形态具象中少数保持半清醒的一个,除 Etya 外第二个知情 NPC |
| 功能 | 她的血是**压制回响的封条**(对应原版树妖净化粉/环境检测的身份);承担中期补充叙事 |
| 树妖之血事件 | 世吞/克脑被击败后,向导(剧本执行者)动了她的血,封条松动 → **血月回归** |
| 与 Etya 的关系 | 肉山后她发现 Etya 对向导的破坏早已知情,两人对质(揭示 Etya 深层动机的演出节点) |

### 5. 玩家(新主角)

- 破封而出(或生于幻象之中)的普通人,不知身处幻象。
- 某天夜里梦见那个人的低语:"你想知道……为什么星星不再坠落,血月不再升起?"——剧情的起点。
- 不是战士,却能前行;这份执着激发了 Etya 残存的使命感。

## 四、叙事结构(四幕)

| 阶段 | 内容 |
|------|------|
| 序章 | 玩家以普通人之身进入平静世界,遇见 Etya |
| 中期 | 通过歌与记忆碎片,逐步了解"传说"的真相,走成长/追寻之路 |
| 终章 | 进入"遗忘尽头"(灰白世界),面对手持天顶剑的终焉者 |
| 结局 | 无论胜负,世界都将进入真正属于"你"的时代 |

**结局分支(三选):**

1. **战胜终焉者**:获得世界重启的权利,进入"未知未来"。
2. **放弃战斗**:真相结局——看见一切,但选择保留和平假象。
3. **加入他**:"你愿意与我一起守望这片虚幻的和平吗?"

## 五、核心玩法系统(叙事绑定)

### 0. 职业解锁(神话隐退)

主角所处的时代神话隐退,魔法职业开局不可用:

| 职业 | 开局状态 | 解锁节点 | 叙事 |
|------|---------|---------|------|
| 战士 / 射手 | 可用 | — | 凡人的武艺不依赖旧神话 |
| 法师 | **锁定**(魔力上限 0,魔法武器不可用) | 击败克眼当晚,天空第一次流星雨,陨星恢复坠落;Etya 传授魔法 | 星星不坠落 = 没有陨星 = 魔力无从谈起;她的歌本来就是旧时代的魔法 |
| 召唤师 | 召唤栏**锁定**;**鞭子可用,吃近战加成** | 击败卡诺斯(第一个梦境副本)后,学会"唤起记忆的声音" | 召唤物 = 回响;解锁后鞭子恢复召唤加成 |

时间线上克眼后正好树妖入住,与魔法解锁节点吻合。

### 1. Boss 框架:幻象与真实两段式(进度节奏总表)

肉山前的 Boss 是幻象回放的"回响",肉山后的 Boss 是入侵现实后实体化的真实存在(世界观解释见第二节"幻象的运作与崩坏")。

| 阶段 | 原版内容 | 模组内容 | 解锁 |
|------|---------|---------|------|
| 序章 | 无星、无血月 | 遇见 Etya,第一首歌指向旧神殿 | 仅战士/射手(鞭子吃近战加成) |
| 前期 | 克眼(第一个回响,幻象) | 掉落首批记忆碎片 | **魔法解锁**,星星回归 |
| 前中期 | 世吞/克脑(幻象)、骷髅王(幻象) | **树妖之血事件**(向导所为)→ 血月回归 | — |
| 肉山前 | — | **梦境副本:卡诺斯**,获"虚言者的傀儡" | **召唤解锁** |
| 困难模式开启 | 肉山(封印守门者) | 幻象表层撕裂演出;树妖与 Etya 对质 | 神圣/腐化 = 幻象崩坏斑块 |
| 困难模式 | 机械三王(实体化的战争兵器记忆) | **万物价主、血医魔母**(现实巢穴 Boss)穿插 | 决意树逐步点开 |
| 后期 | 世纪之花、石巨人、教徒、柱子(最终战争的回放) | 三件旧物集齐 → 三次共鸣、记忆之井 | — |
| 月总 | **月球领主 = 重演终焉者最后一战(真实)** | 真相完全揭示 | 解锁"终焉之路" |
| 终章 | — | 遗忘尽头、终焉者三阶段 | 三结局 |

平衡注意:魔法解锁前的所有内容必须纯战士/射手可通关;若在克眼前追加模组 Boss,同样要保证这一点。

### 2. 歌与记忆系统(Etya 核心交互)

- Etya 每首歌对应一个地点、一个 Boss 或一段历史,以暗示而非说明书的方式引导。
- **肉山前**:玩家收集**记忆/残响碎片**(原版 Boss 即回响,击败后掉落)交给她,拼成一段记忆 → 她唱歌 → 开启**梦境副本**(卡诺斯)。
- **肉山后**:歌的功能从"开门"变为**定位**——她听见实体化回响的位置,歌指向万物价主、血医魔母的现实巢穴。碎片提交循环不变。
- 引导示例:

| 玩家状态 | 她的歌 | 引导效果 |
|----------|--------|----------|
| 初到世界 | 《远方的钟声》 | 指向旧神殿废墟,第一个试炼 Boss |
| 击败某 Boss | 《熄灭的炉火》 | 指向被遗忘的城镇角落 |
| 达到某战力 | 《断剑之歌》 | 揭露天顶剑与最终目标 |
| 暴力路线 | 歌调忧伤、歌词模糊 | 她动摇,Bad Ending 前兆 |

### 3. 灵魂共鸣(Resonance)

- 玩家首次获得"旧遗物"时,她会说:"它在回应你……"
- 佩戴 3 件旧物后,她唱出完整之歌,带玩家前往**记忆之井**(大剧情解锁)。
- 三次共鸣完成后,解锁"终焉之路"提示,主线推进至终章。

### 4. 成长系统

- 战斗胜利后获得"能力觉醒",例如:闪避机制(幻影步)、废弃物组合武器、"余火槽"(蓄力换技能)。
- **"决意"系统 = 四职业天赋树**(战士/射手/法师/召唤四棵大树):
  - 每棵树下 2~3 个小流派,如战士下分"血之信仰"暴击流、"余火之道"耐力反击流。
  - 天赋点绑剧情节点(Boss 首杀、共鸣、能力觉醒),**不靠刷怪攒点**——成长来自记忆继承。
  - 已有的能力觉醒(幻影步等)做成树中的公共节点。
  - 工程量大,实装后置(M4 之后),但存档字段(天赋点、已点节点)在 M2 做进度系统时预留。

### 5. Etya 的商店

| 类型 | 描述 |
|------|------|
| 旧 NPC 遗物 | 护士的手套、向导的徽章、渔夫的鱼饵盒等,带微弱异能 |
| 追忆物品 | 解锁"记忆残响"副本的钥匙,如"护符残片·向导" |
| 碎片卷轴 | 解锁技能、能力或提示 |

### 6. 终焉者三阶段 Boss 战

1. **第一阶段**:招式全部为旧 Boss 融合(地牢守卫冲撞 + 鱼龙突袭 + 星尘龙等)。
2. **第二阶段**:完全拟人战斗(类玩家 PvP),天顶剑、星尘龙召唤、泰拉刃组合连击。
3. **第三阶段**:领域变化,世界变为灰白;玩家失去 UI,靠声音和影像判断攻击时机。
- 战场:专属场景"记忆的尽头"/"英雄之殿",黑白风格 + 专属音乐《余火残响》。

## 六、氛围细节(可选彩蛋)

- 旧时代归隐 NPC 的新对话:"我常梦见他站在城镇的中央,那时的阳光真好。"
- Etya 身边的小猫是前代主角宠物的转世,名叫「MeowGuard」。
- 终焉者战败后,主城留下雕像;碑文可致敬 Hetia 废案:"梦的守火者"。
